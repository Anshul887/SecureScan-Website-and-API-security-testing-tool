# SecureScan

Website and API Security Testing Platform

## Features

- Authentication
- Security Scanning
- Vulnerability Detection
- PDF Reports
- Scan History
- Dashboard Analytics

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind

Backend:
- Node.js
- Express
- Prisma

Database:
- PostgreSQL

Infrastructure:
- Docker
- Redis

## Setup

Start services:

docker compose up -d

Backend:

cd backend
npm install

Frontend:

cd frontend
npm install

Run:

npm run dev
